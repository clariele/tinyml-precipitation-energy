# TinyML Precipitation Energy

Code accompanying the paper:

"Performance and Energy Efficiency Analysis of TinyML Models for Precipitation Forecasting on Edge Devices"

## Models
- TinyMLP
- TinyCNN

## Variants
- Baseline
- Pruned
- Quantized
- Knowledge Distillation

## Hardware
- Arduino Nano 33 BLE Sense
- INA219 current sensor

## Repository structure
- tinyMLP/: MLP models and Arduino inference code
- tinyCNN/: CNN models and Arduino inference code
