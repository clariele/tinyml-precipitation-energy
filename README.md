# TinyML Precipitation Energy

Code accompanying the paper:

"Avaliação de Desempenho e Consumo Energético de TinyML em Dispositivos de Borda para Previsão de Precipitação"

## Models
- TinyMLP
- TinyCNN

## Variants
- Baseline
- Pruned
- Quantized
- Knowledge Distillation

## Hardware
- Arduino Nano 33 BLE Sense
- INA219 current sensor

## Repository structure
- tinyMLP/: MLP models and Arduino inference code
- tinyCNN/: CNN models and Arduino inference code
